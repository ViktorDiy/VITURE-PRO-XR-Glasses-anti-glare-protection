# VITURE-PRO-XR-Glasses-anti-glare-protection
VITURE PRO XR Glasses anti-glare protection

I often take my dog ​​for walks in nature while learning a foreign language using inexpensive VITURE PRO XP Glasses. However, glare on the lenses and screens is a significant hindrance.<br/>
To improve this, I developed an anti-glare protection that significantly reduces glare without significantly interfering with orientation while walking.<br/> 
The protection is easy to install on the glasses. To do this, remove the nosepiece (I use the smallest one) and then, applying slight pressure, insert the left corner of the nosepiece into the glasses first (for the left nosepiece, when looking at the glasses from the screen side).<br/> 
The anti-glare protection is printed using PETG filament. The orientation of the parts during printing as shown in the photo worked best for me.<br/>

You can download 3D printing files from my repository.<br/>
